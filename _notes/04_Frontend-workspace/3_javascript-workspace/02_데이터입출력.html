* 자바스크립트에서의 데이터 입출력(입출력 구문)

1. 데이터를 출력한느 기본구문(출력문들)

1) window.alert("알림창에 출력할 문구");
2) window.console.log("콘솔창에 출력할 문구");
3) document.write("화면에 출력할 문구");
4) 선택한요소.innerHTML or .innerText="해당요소에 출력할 문구";


window.alert("알림창에 출력할 문구");


window.console.log("출력창에 출력할 문구");


document.write("화면에 출력할 문구")


선택한 요소.innerHTML / .innerText="해당요소에 출력할 문구"

자바스크립트에서 어떤 요소(element) 안의 값(content 영역의 값)을 가지고 온다거나, 변경하고자 할 때 innerHTML, innerText아는 속성에 접근해서 처리할 수 있다.

2. 데이터를 입력받는 기본구문(입력문들)

1) 변수 = window.confirm("질문내용");
2) 변수 = prompt("질문내용");
3) 변수 = 선택한요소.속성명(id/className/innerHTML/innerText..);
4) 변수 = 선택한input요소.value;


1) 변수 = window.confirm("질문내용");

- confirm 호출시 질문내용과 확인/취소 버튼이 존재하는 알림창 발생
- 확인버튼 클릭시 true 취소시 false 반환


2) 변수 = window.prompt("질문내용");

- prompt 호출시 질문내용과 입력할 수 있는 텍스트상자와 확인/취소 버튼이 있는 알림창 발생
- 확인버튼 클릭시 텍스트상자에 입력되어있는 값이 반환
- 취소버튼 클릭시 null 반환
- 암것도 입력안하고 확인버튼 누르면 빈문자열 "" 반환



3) 변수 = 선택한요소.속성명;
